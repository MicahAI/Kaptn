---
description: Reset Kaptn limits, loop history, and pauses
---

Run this command with Bash and confirm the result to the user:

```bash
"$CLAUDE_PLUGIN_ROOT/scripts/kaptn-ctl" reset
```

If `CLAUDE_PLUGIN_ROOT` is not set in the Bash environment, locate the
kaptn plugin directory under `~/.claude/plugins/` and run
`<plugin-dir>/scripts/kaptn-ctl reset` instead.
